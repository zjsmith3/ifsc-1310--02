Thoughts on Progressive Enhancement

Progressive Enhancement is a way to design and create websites with the thought of including everyone, no matter the browser. Having the mindset of making sure the content is there, and well delivered, so no matter what browser, the user can digest the content. Then you can add a layer of css. This makes the website deliver content, but also look good while doing it. Finally, there is the awesome last layer of css. This is where you can add pizzazz to your website. Awesome code that adds a layer of cool features that wouldn’t be possible without. To put it plainly, like https://www.freecodecamp.org/news/what-is-progressive-enhancement-and-why-it-matters-e80c7aaf834a/ This article did, …it gives us basic user experience and cross compatibility across browsers to ensure stability. Now this doesn’t mean that you it’s impossible to make your website look good while having it accessible to all the web browsers. According to the linked article, your still able to use things like cool fonts and features. It’s just good to make sure you check to make sure that your website works without those features, and that in general the majority of people can use your website. In conclusion Progressive Enhancement is important all around the internet to make it a better time for anyone, no matter what kind of browser you use.

Web Accessibility writing

From watching the video and doing research, I can say that web accessibility doesn’t just stem from wanting to make your websites more accessible to those who aren't lucky enough to have normal functionality, but also shows the quality of work that someone does on the website. To have your site be accessible for everyone it first has to be created in a well manner. Like using progressive enhancements. Meaning using HTML for core content and behavior, css for presentation, and javascript for advanced behavior. This also means using universal behaviors in symbols. Like yellow triangles mean warnings and green check marks are good. There’s many layers when it comes to web accessibility that people can and do take to make the internet an easier place to traverse. And the interesting thing is that web accessibility tends to make sites more successful. This is due to the fact that when a site is accessible, that means it is well made, meaning more people will share it. So there’s more interest in making accessible sites than just for the good of the people. In the end of the day web accessibility is a very interesting topic with many layers, and personally holds importance for all the advantages it gives sites, like more publicity, success.
